The project includes the following subtasks:
- Data cleaning involves handling missing values, ensuring correct datatypes, filtering for cars from 1995 onwards, and standardising text entries in specific column.
- Feature Engineeering involves creating new columns for analysis: calculating average city and highway MPG as Total MPG and determine price per hP by dividing MSRP by Engine HP. (i.e., Creating new columns, removing duplicates and dealing with missing values)
- Exploratory Data Analysis (EDA) involves exploring dataset using descriptive statistics, group-based analysis, and various visualizations to uncover patterns. Additionally perform a correlation analysis to understand relatioships between key variables.
- Data Precesing
- Encoding Categorical Columns
- Scaling
- Building the Machine Learning (Regreesion model) to predict the prize of cars (train and test dataset)
- Validating and evaluating the model
- Checking the Performance metrics
